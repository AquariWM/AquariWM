Module xrb::proto::atoms

source · [−]
Expand description

Atom constants that are predefined in the X11 protocol.

The X11 protocol predefines the values of the following 68 Atoms for convenience. These Atoms always have the same value, across all libraries, all connections, and all clients. Other Atoms, i.e. Atoms not listed here, and which are not predefined, are dependent on the connection: the only way to know what these other Atoms’ values are for any given connection is to send InternAtom requests for them.

Once one InternAtom reply has been received, however, you can assume it will remain the same for the duration of the current connection.
